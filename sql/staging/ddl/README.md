### DDL
DDL stands for **Data Definition Language**. It is used to define and manage database structures such as tables, columns, keys, and constraints.

### `ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_0900_ai_ci`
This sets the table to use the InnoDB storage engine and stores text in utf8mb4 with the specified collation for character comparison and sorting.
